# Bullzilla Presale DApp

This project includes:
- React Frontend
- Solidity Smart Contracts
- Hardhat Deployment

## Installation

```bash
npm install
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

## Deploy Smart Contracts

```bash
npx hardhat compile
npx hardhat run scripts/deploy.js --network goerli
```
